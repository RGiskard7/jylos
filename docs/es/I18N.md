# Internacionalización

English: [../I18N.md](../I18N.md)

Jylos soporta inglés y español mediante resource bundles.

## Archivos

```text
jylos/src/main/resources/com/example/jylos/i18n/messages.properties
jylos/src/main/resources/com/example/jylos/i18n/messages_es.properties
```

`messages.properties` es inglés/base. `messages_es.properties` es español.

## Añadir idioma

1. Crear `messages_<locale>.properties`.
2. Copiar todas las claves de `messages.properties`.
3. Traducir valores, no claves.
4. Añadir opción de idioma en UI si procede.
5. Probar menús, diálogos, command palette y errores visibles.
6. Actualizar README/docs si el idioma pasa a estar soportado.

## Reglas

- No hardcodear textos visibles en controladores.
- Mantener claves estables.
- Si falta traducción, debe existir fallback razonable.
- Command palette, tooltips y diálogos también deben estar traducidos.
